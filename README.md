# Projeto_Final_Pos_Senai_ML
O desafio proposto pela ONG "Elfos Muito Legais" foi desenvolver um modelo capaz de classificar lâminas de sangue para diagnosticar a fictícia "Doença de Sauron".  Exploro duas estratégias distintas para resolver este problema de classificação binária (Positivo vs. Negativo) utilizando um dataset de 27 mil imagens.
